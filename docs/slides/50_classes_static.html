Les Classes

Les classe et les méthodes statiques

NOus avons la possibilité de définir comment accéder à nos propriétés

    
      class PersonClass {
        constructor(name) {
          this.name = name;
        }
        sayName() {
          console.log(this.name);
        }
        static create(name) {
          return new PersonClass(name);
        }
      }
      let person = PersonClass.create("Nicholas");
  • Les propriétés statiques sont des éléments de classes et pas d'instance
