ES6 ?

ECMA International = European Computer Manufacturers Association
-> tente de standardiser les ordinateurs (technologie, sécurité, stockage, structure, langage, etc.)

Lors de la bataille Netscape Microsoft pour le script, Netscape a soumis son Javascript à l'ECMA afin d'en faire le standard.

Depuis 2015 nous sommes sur ES2015 appelé ES6. DONC: Javascript = langage / ECMA script = Standard/norme
