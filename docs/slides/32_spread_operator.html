C'est quoi le spread operator ??

Il permet de splitter les valeurs multiples en valeur simple et de les utiliser 

        
                let a = [2, 3, 4];
                let b = [1, ...a, 5];
                console.log(b);         // [1, 2, 3, 4, 5]
