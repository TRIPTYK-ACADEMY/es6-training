Rafraîchissons notre savoir sur le scope

        
        function setWidth(){
                var width= 100;
                console.log(width); 
        }
        setWidth();
        console.log(width);
Si var width, n'existe que dans function, le 2e console log ne récupère rien. Si width sans var le 2e console log récupère "window.width" . Jusqu'à ES5 pas de gestion du scope.
