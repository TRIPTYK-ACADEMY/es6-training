Les promises

Comment faire pour attendre la fin de multiple promesses

Promise.all([Promises]) permùet d'attendre la fin de toutes promesses et ensuite de faire le then

    
      var promise1 = Promise.resolve(3);
      var promise2 = 42;
      var promise3 = new Promise(function(resolve, reject) {
        setTimeout(resolve, 100, 'foo');
      });
      
      Promise.all([promise1, promise2, promise3]).then(function(values) {
        console.log(values);
      });
      // expected output: Array [3, 42, "foo"]
