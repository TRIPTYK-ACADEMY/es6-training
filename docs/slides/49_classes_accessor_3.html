Les Classes

Les classe et les accessors

NOus avons la possibilité de définir comment accéder à nos propriétés

    
      class Warrior {
        constructor(name) {
          this._name = name;
          this._health = 100;
        }
        get health() {
          return this._health;
        }
      }
      let donatello = new Warrior("Donatello");
      console.log(donatello.health);   // 100
      donatello.health = 80;   // TypeError: Cannot set property health...
  • Quand on ne défini pas le setter, la propriété devient read-only
