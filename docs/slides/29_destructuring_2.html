La destruction en javascript

La destructuration permet de travailler sur les fonctions. 

        
            function greet({name: greeting = 'she', age}) {
                console.log(`${greeting} is ${age} years old.`)
            }
            greet({name: 'nico', age: 27});
            // nico is 27 years old
            greet({age: 24});
            // she is 24 years old
