Les modules

Comment importer les modules !!!

Pour importer toutes les valeurs

    
        import * as example from "./example.js";
        console.log(example.sum(1, example.magicNumber));
        console.log(example.multiply(1, 2));
  • Dans ce cas on importe bien avec les {} pour définir ce que l'on importe ou pas
  • Attention qu'importe le fois que vous importez le module, il ne sera instantié qu'une suele fois
