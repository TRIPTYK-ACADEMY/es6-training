La destruction en javascript

La destructuration permet de récupérer des valeurs d'objets ou de array complexe simplement. 

        
           let expense = {
                 type: 'business',
                 amount: '€ 50.00'
               };
               let type = expense.type;
               let amount = expense.amount;
             // Destructuring in es6
               let { type, amount } = expense;
               type = 'family';
               console.log(type);
            console.log(amount);
            console.log(expense);
            let [x, y] = ['Jane', 'Leo'];
            x = 'Jane', y = 'Leo'
