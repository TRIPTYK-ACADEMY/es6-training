Les Classes

Les classe et l'héritage

    
      class Warrior {
        constructor(name) {
          this.name = name;
          this.health = 100;
        }
        heal(amount) {
          this.health = Math.min(this.health + amount, 100);
        }
      }
      class Ninja extends Warrior {
        constructor(name) {
          super(name);
        }
        heal(amount) {
          super.heal(amount * 1.2);
        }
      }

  • Si une classe nécessite d'instancier son parent avec des valeur, le constructeur doit utiliser super() pour le faire
  • Attention que vous devez utiliser super avant de pouvoir utiliser this dans le constructeur
